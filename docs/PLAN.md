# MiLedgera Frontend — Implementation Plan

This plan is split into three parts, each covering two phases. Every
checklist task below is tracked as its own GitHub issue (see
[CONTRIBUTING.md](../CONTRIBUTING.md)).

Sibling repos referenced below:
- Contract — https://github.com/MiHashport/MiHashport-Contract
- Backend — https://github.com/MiHashport/MiHashport-Backend

## Parts

| Part | Phases | Focus |
|---|---|---|
| [Part 1](PLAN-part-1.md) | 1-2 | Foundation hardening, transaction input & wallet connection |
| [Part 2](PLAN-part-2.md) | 3-4 | Backend integration, verdict & evidence display |
| [Part 3](PLAN-part-3.md) | 5-6 | Delivery proof & history, agent integration & polish |

Phase 1 (Foundation) scaffold is complete; remaining Phase 1 issues are
hardening/infra follow-ups. Phases are ordered so the app stays
demoable as early as possible.
